# 🌱 IoT-Based Smart Plant Monitoring and Control System

## 📌 Description
This project is an IoT-based smart plant monitoring and control system. 
It monitors soil moisture, temperature, and humidity, and automatically controls watering using a water pump.

## 🔧 Components Used
- NodeMCU (ESP8266)
- Soil Moisture Sensor
- DHT11 Sensor
- Relay Module
- Water Pump
- Power Supply
- Blynk IoT Cloud
- Plants
- Mobile user

## ⚙️ Features
- Automatic plant watering system
- Real-time monitoring using IoT
- Remote control using mobile (Blynk app)
- Efficient water usage
- Suitable for home gardening and small agriculture
